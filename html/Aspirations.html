Future Aspirations

As I am currently in the Common ICT programme, this indicates I have multiple specialisations to choose from which allows me to explore different career paths. However, regardless of the direction I head towards, I will continue to hone my computing skills and embark on a path of continuous learning.

Digital Forensics Investigator

Digital Forensic Investigators carry out criminal investigations by analyzing the structure of a digital attack.Using via backtracing or deleted messages and emails,a digital forensic investigator can uncover deleted history.

Digital Forensics Investigator

Skills Criteria:

  1. High speed of analytical thinking with precise observation abilities
  2. Knowledge of computer networking
  3. Accessing various operating systems
  4. Understanding of the latest breaches, threats, and vulnerabilities
  5. Willingness to learn and constantly upgrade to the latest trends and technologies

Data Analyst

Data Analysts implement data analytics tools to identify trends within the international markets. As someone who enjoys Mathematics and Computing, being a data analyst would be a very good prospect.

Data Analyst

Skills Criteria:

  1. Visualising data in high-quality charts and graphs to present findings in a clear and concise way
  2. Coherence, storage and availability of data statistics, to meet an organisation's needs
  3. Formulating different patterns within the data by the implementation of computational algorithms
  4. Critical thinking
  5. Using algorithms and statistical models to analyse and draw inferences from patterns in data via machine learning

Artificial Intelligence Engineer

An A.I engineer creates A.I models utilizing deep learning neural networks and machine learning algorithms in order to derive business insights that may be used to business decision-making. It is very fascinating to communciate with A.I and hopefully I am able to create a model one day.

Artificial Intelligence Engineer

Skills Criteria:

  1. Optimize system performance and enable the scalability of A.I Applications
  2. Evaluate and benchmark machine learning solutions to integrate them into existing systems
  3. Design, develop, and deploy AI systems
  4. Experience with cloud computing
  5. Good communication skills

Full Stack Developer

A Full Stack Developer creates applications using a variety of technologies and languages, such as front end, that is responsible for the website's visual look, and back end which handles the site's architecture and logic.

Full Stack Developer

Skills Criteria:

  1. Front-end Languages and Frameworks
  2. Fundamental design skills like UI/UX design
  3. Enhance visible features of the app with which users interact with
  4. Soft Skills(Problem-solving, Analytical skills, Strategic planning and Creativity)
  5. Familiar with web architecture basics, code structure and where computational duties are located
